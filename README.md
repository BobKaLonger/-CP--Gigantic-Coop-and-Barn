Is mega still not enough? Do you want to have a silly amount of animals in one building? Well, for some reason, I did, so I made this mod.

Gigantic Coops and Barns will hold up to 40 animals each, with TWO rows of hay and plenty of room for some machines. I made the maps extremely clean (no hay piles or junk) with a regular floor and lighting pattern because of my personal preference. There is now a new config option to add some hay and a few objects if you prefer how it looks! The water sources ARE water sources, and the Giga Coop will now come with TWO extra incubators.

The building footprints and exterior appearance stay the same as the vanilla Deluxe Coop and Barn (just like the Mega Coop and Barn) so any recolors or reskins should carry over. These buildings won't take up any more space but you're going to need more silos....or just a Bigger Silo Size.
